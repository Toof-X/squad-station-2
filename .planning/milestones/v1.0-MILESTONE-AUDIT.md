### Integration Finding: HOOK-02 Configuration Gap

Hook scripts (`hooks/claude-code.sh`, `hooks/gemini-cli.sh`) exist and are correct but are not registered in `.claude/settings.json` (missing `Stop` event) or `.gemini/settings.json` (missing `AfterAgent` event). Phase 2 SUMMARY (02-02) documents this as "User Setup Required" — manual hook registration is a deployment concern. The scripts themselves satisfy HOOK-02.

### Design Note: ui.rs DB Pattern Divergence

`ui.rs` constructs its own read-only `SqlitePool` instead of calling `db::connect()`, omitting `busy_timeout` and `create_if_missing`. This is intentional per the 03-02 SUMMARY's "connect-per-refresh" strategy to prevent WAL checkpoint starvation. On DB error, the TUI silently retains stale data. WAL mode and `max_connections(1)` are correctly applied.

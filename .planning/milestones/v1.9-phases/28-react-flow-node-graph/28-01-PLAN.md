**Step 3: Create `web/src/hooks/useGraphLayout.ts`**

Hook that transforms `agents[]` and `messages[]` into positioned React Flow `Node[]` and `Edge[]`.

- Import `dagre` from `@dagrejs/dagre`, and `Node`, `Edge` from `@xyflow/react`.
- Import `Agent`, `WsMessage` from `./useSquadWebSocket`.
- Export `AgentNodeData` type (or re-export from AgentNode).

**`getLayoutedElements` helper function** (NOT a hook, pure function):
- Create dagre graph: `new dagre.graphlib.Graph().setDefaultEdgeLabel(() => ({}))`.
- Configure: `g.setGraph({ rankdir: 'TB', ranksep: 120, nodesep: 80 })`.
- Set each node with `{ width: 150, height: 60 }`.
- Set edges.
- Run `dagre.layout(g)`.
- Map nodes back with `position: { x: pos.x - 75, y: pos.y - 30 }` (center offset).
- Set `targetPosition: 'top'`, `sourcePosition: 'bottom'` on each node.

**`useGraphLayout(agents, messages)` hook**:
- **Orchestrator detection**: Find agent whose `role` contains "orchestrator" (case-insensitive). Fallback: agent appearing most as `from_agent` in messages. Final fallback: first agent.
- **`useMemo` for node creation** (keyed on `JSON.stringify(agents)`):
  - Map each agent to a React Flow Node of type `'agent'` with `AgentNodeData` in `data` field.
  - Set `data.isOrchestrator` for the detected orchestrator.
- **Edge creation**: Create a structural edge from orchestrator to each non-orchestrator agent. Edge id: `e-{orchestrator.name}-{worker.name}`. Edge type: `'animated'` (will be used by Plan 02's custom edge; for now, default edge type is fine — set `type: 'default'`).
  - For each edge, check if there's a message with `status === "processing"` where `from_agent` matches orchestrator name and `to_agent` matches worker name (or vice versa). If so, set `animated: true` on the edge data and include `task`, `priority`, `timestamp` from the message.
- **Layout**: Run `getLayoutedElements(nodes, edges)` inside a `useMemo` keyed on agent IDs (NOT full agent objects — avoid re-layout on status-only changes). Use `agents.map(a => a.name).sort().join(',')` as the layout key.
- **Edge styling update**: A SEPARATE `useMemo` keyed on `messages` that updates edge `data.animated`, `data.task`, `data.priority`, `data.timestamp` without re-running dagre layout.
- Return `{ nodes, edges }`.

Export the `AgentNodeData` type and `useGraphLayout` hook.

**Step 1: Create `web/src/components/AnimatedEdge.tsx`**

Custom React Flow edge component with SVG `<animateMotion>` crawling dots and `EdgeLabelRenderer` for in-flight message info.

- Import `BaseEdge`, `EdgeLabelRenderer`, `getSmoothStepPath` from `@xyflow/react`.
- Import `EdgeProps`, `Edge` from `@xyflow/react`.

**Type definition:**
```typescript
type AnimatedEdgeData = {
  animated: boolean;
  task?: string;
  priority?: string;
  timestamp?: string;
};
type MessageEdge = Edge<AnimatedEdgeData, 'animated'>;
```

**Component `AnimatedEdge`** (exported, accepts `EdgeProps<MessageEdge>`):
- Destructure: `id, sourceX, sourceY, targetX, targetY, sourcePosition, targetPosition, data`.
- Calculate path: `const [edgePath, labelX, labelY] = getSmoothStepPath({ sourceX, sourceY, sourcePosition, targetX, targetY, targetPosition });`
- Render:
  1. `<BaseEdge id={id} path={edgePath} />` — the base edge line.
  2. **Crawling dots** (only when `data?.animated` is true):
     - Render 3 SVG `<circle>` elements with `r="3"` and `fill="#3b82f6"` (blue-500).
     - Each circle contains `<animateMotion>` with:
       - `dur="2s"`, `repeatCount="indefinite"`, `path={edgePath}`
       - Staggered `begin`: `"0s"`, `"0.66s"`, `"1.33s"` (evenly spaced across duration).
     - This creates the crawling dots effect per research findings — GPU-accelerated via SVG.
  3. **Edge label** (only when `data?.task` exists — i.e., in-flight message):
     - Use `<EdgeLabelRenderer>` component.
     - Render a `<div>` positioned at the edge midpoint:
       ```
       style={{
         position: 'absolute',
         transform: `translate(-50%, -50%) translate(${labelX}px, ${labelY}px)`,
         pointerEvents: 'all',
       }}
       className="nodrag nopan bg-gray-800 dark:bg-gray-800 border border-gray-600
                  rounded px-2 py-1 text-xs shadow-md"
       ```
     - Content:
       - Task text: truncated to 30 chars with ellipsis. `text-gray-200 font-medium`.
       - Priority badge: small colored badge — "urgent" in red, "high" in orange, "normal" in gray. Use a lookup.
       - Timestamp: relative time from `data.timestamp` (e.g., "2m ago"). Use a simple helper: calculate difference from `new Date()` and format as Xm/Xh ago. If no timestamp, omit.

**Step 1: Create `web/src/hooks/useTheme.ts`**

- Export `type Theme = 'dark' | 'light';`
- Export `useTheme()` hook:
  - `useState<Theme>` initialized from `localStorage.getItem('squad-theme')` or default `'dark'`.
  - `useEffect` syncs theme to DOM: add/remove `.dark` class on `document.documentElement`.
  - `useEffect` persists to localStorage on theme change: `localStorage.setItem('squad-theme', theme)`.
  - Returns `{ theme, toggleTheme }` where `toggleTheme` flips dark<->light.

**Step 2: Create `web/src/components/ThemeToggle.tsx`**

- Accept props: `{ theme: Theme; onToggle: () => void }`.
- Render a button in top-right corner area (will be placed by App.tsx):
  - Show sun icon (Unicode `\u2600` or SVG) when dark (clicking switches to light).
  - Show moon icon (Unicode `\u263D` or SVG) when light (clicking switches to dark).
  - Styling: `p-2 rounded hover:bg-gray-700 dark:hover:bg-gray-700 text-gray-400 dark:text-gray-400 transition-colors`.
  - `aria-label="Toggle theme"` for accessibility.

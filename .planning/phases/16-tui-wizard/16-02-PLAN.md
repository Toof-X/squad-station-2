```rust
// Check if squad.yml exists; if not, run the interactive wizard
if !config_path.exists() {
    match crate::commands::wizard::run().await? {
        Some(result) => {
            // Phase 17 will use result to generate squad.yml and continue init
            // For Phase 16: print result summary and return
            println!("Wizard completed:");
            println!("  Project: {}", result.project);
            for (i, agent) in result.agents.iter().enumerate() {
                println!(
                    "  Agent {}: role={}, tool={}, model={}, desc={}",
                    i + 1,
                    agent.role,
                    agent.tool,
                    agent.model.as_deref().unwrap_or("-"),
                    agent.description.as_deref().unwrap_or("-"),
                );
            }
            println!("\n(squad.yml generation will be added in Phase 17)");
            return Ok(());
        }
        None => {
            println!("Init cancelled.");
            return Ok(());
        }
    }
}
```

This block:
1. Checks `config_path.exists()` — if squad.yml is present, falls through to existing `load_config` flow (no behavior change for existing users).
2. If no squad.yml: calls `wizard::run().await?` to launch the TUI.
3. On `Some(result)`: prints a summary of collected values and returns. Phase 17 will replace this print block with actual squad.yml generation.
4. On `None` (Ctrl+C): prints "Init cancelled." and returns cleanly.

Do NOT modify any other part of init.rs. The existing init flow for when squad.yml already exists must remain unchanged.

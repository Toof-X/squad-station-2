       name: &str,

   ) -> anyhow::Result<()> {
       let now = chrono::Utc::now().to_rfc3339();
       sqlx::query(
           "UPDATE agents SET status = ?, status_updated_at = ? WHERE name = ?"

       .bind(&now)
       .bind(name)
       .execute(pool)
       .await?;
       Ok(())
   }

Rewrite `src/commands/signal.rs` to add the 4-layer guard BEFORE the existing signal flow. The guard order is critical (from CONTEXT.md locked decision):

**Guard 1 -- Not in tmux (HOOK-03):** Check `std::env::var("TMUX_PANE")`. If `Err` (not set), return `Ok(())` silently. This is the cheapest check and catches invocations outside tmux.

**Guard 2 -- Config/DB connection failure:** Wrap `config::load_config()` and `db::connect()` in match blocks. On error, print warning to stderr (`eprintln!("squad-station: warning: {e}")`) and return `Ok(())`. Per locked decision: real errors go to stderr but NEVER fail the provider (exit 0 always).

**Guard 3 -- Agent not registered (HOOK-03):** After connecting to DB, call `db::agents::get_agent(&pool, &agent)`. If `None`, return `Ok(())` silently. CRITICAL: This replaces the current `bail!("Agent not found: {}", agent)` on line 18-19. The bail is wrong for hook context -- unregistered agents must be silent exit 0.

**Guard 4 -- Orchestrator self-signal (HOOK-01):** If `agent_record.role == "orchestrator"`, return `Ok(())` silently. This prevents the infinite loop where orchestrator's hook signals itself.

**After all guards pass:** Execute the existing signal flow (update message status, notify orchestrator). Additionally, after a successful signal (rows > 0), call `db::agents::update_agent_status(&pool, &agent, "idle")` to set the agent back to idle.

Also update the `send` command to set agent status to "busy" when a task is sent: Add `db::agents::update_agent_status(&pool, &agent, "busy").await?;` after the message insert in `src/commands/send.rs`. This keeps agent status accurate across the send->signal lifecycle.

    let config_path = std::path::Path::new("squad.yml");
    let config = match config::load_config(config_path) {
        Ok(c) => c,
        Err(e) => {
            eprintln!("squad-station: warning: {e}");
            return Ok(());
        }
    };
    let db_path = match config::resolve_db_path(&config) {
        Ok(p) => p,

            return Ok(());
        }

        Err(e) => {
